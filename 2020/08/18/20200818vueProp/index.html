单向数据流 One-Way Data Flow

父级 prop 的更新影响子组件，反之不行，不能从子组件更新父组件。如果在子组件内部改变prop，Vue 会在浏览器的控制台中发出警告。
每次父级组件发生变更时，子组件中所有的 prop 都将会刷新为最新的值。

非 Prop 的 Attribute

一个非 prop 的 attribute 是指传向一个组件，但是该组件并没有相应 prop 定义的 attribute。

因为显式定义的 prop 适用于向一个子组件传入信息，然而组件库作者不能预见组件会被用的所有场景。所以组件可以接受任意的 attribute，而这些 attribute 会被添加到这个组件的根元素上。

例如，通过一个 Bootstrap 插件使用了一个第三方的 <bootstrap-date-input> 组件，这个插件需要在其 <input> 上用到一个 data-date-picker attribute。可以将这个 attribute 添加到你的组件实例上：

这个 data-date-picker="activated" attribute 就会自动添加到 <bootstrap-date-input> 的根元素上。
